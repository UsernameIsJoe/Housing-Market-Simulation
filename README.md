# Market Simulation

## intro

This project develops a lightweight agent-based simulation that models interactions between heterogeneous agents and a simplified market environment.

The system focuses on how individual decision rules and agent attributes shape market dynamics over time. Agents evaluate available options based on census-based data preferences and constraints, allowing the simulation to explore how micro-level behaviors aggregate into macro-level outcomes.

The framework is designed as a modular prototype that supports experimentation with different agent types, decision rules, and environmental conditions.

## Setup

The project is implemented in Python and organized into modular scripts for agents, environment, and simulation logic.

After cloning the repository, install the required dependencies and run the main simulation script to start the model. The simulation parameters can be adjusted within the configuration files to experiment with different scenarios.

## What's Next

Future development will focus on expanding agent behaviors and improving the realism of market interactions. Planned extensions include richer agent attributes, additional decision mechanisms, and more complex environmental dynamics to support larger-scale simulations.
